Network 2: Calculadora de IPs
Este módulo tiene como objetivo realizar un script que calcule todas las IPs posibles recibiendo como entrada una dirección IP y una máscara. La salida es en binario.
En este caso se calculan todas las IPs posibles para 192.168.8.23 con máscara 255.0.250.255.
