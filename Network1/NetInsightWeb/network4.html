Network 4: Escaneador de red.
Network 4 tiene como objetivo escanear la red para detectar qué IPs están activas, escanear todos los puertos y ver su estado, y ver cuál es el servicio más probable de ejecución. Todo se esto se realiza con la herramienta "nmap". Pulsando el botón verás un ejemplo real de uso realizado para esta actividad, como siempre en mi red local.

Ver terminal 

nmap -sU -p 1-65535 -oN 192.168.0.20-UDP.txt 192.168.0.20
